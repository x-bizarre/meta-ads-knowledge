# 02 — Audiences and targeting (Meta Ads, B2C + traffic, 2025-2026)

---

### Rule 2.1: Broad + Advantage+ Audiences — the new default
- **Gist:** Meta is pushing everyone toward broad targeting. Advantage+ Audience (A+A) is on by default. The algorithm finds buyers across the whole ecosystem on its own, using pixel/conversion signals rather than manual interests.
- **Meta's claim:** CPA is lower by 14.8% (top of funnel), 9.7% (middle), 7.2% (bottom) vs regular audiences. ⚠️ These are Meta's own figures (self-promotion) — don't cite them to a client as a guarantee.
- **Reliability:** [OFFICIAL META] (the mechanics) + self-promotion (the figures)
- **Source:** https://www.facebook.com/business/ads/meta-advantage-plus/audience

### Rule 2.2: Targeting inputs = "hints," not rules
- **Gist:** Inside A+A, interests, age, gender, lookalike, and custom audiences act as RECOMMENDATIONS. The only hard constraints are: geo, minimum age, language, and exclusions (excluded custom audiences).
- **Reliability:** [OFFICIAL META]

### Rule 2.3: Whether hints get priority — DISPUTED
- **Gist:** Meta says "hints are prioritized." In his tests, Jon Loomer found the opposite: with hints, reach into the target segment was actually LOWER. Takeaway: hints are a weak, unreliable lever — don't build a strategy on them.
- **Reliability:** [DISPUTED] — Meta vs independent tests
- **Source:** https://www.jonloomer.com/meta-ads-targeting-2026/

### Rule 2.4: Lookalikes are alive, but not central
- **Gist:** LALs still work, but Meta itself builds lookalike-style expansions inside A+A. Use LALs as a seed for tests and where you need control (lead gen, new markets), not as the foundation.
- **Reliability:** [AGENCY CONSENSUS]

### Rule 2.5: Lookalike percentage — test it, don't dogmatize
- **Gist:** 1% = closer but burns out fast; 3-5% = more durable at scale. Start: **2-3%** for conversions, **5%** for awareness; test 1% and 3-5% in parallel. "Always 1%" is an outdated dogma. For small countries, 1% can be too small.
- **Reliability:** [AGENCY CONSENSUS]

### Rule 2.6: Source size for Lookalikes
- **Gist:** The technical minimum is 100 people, but that's not a workable minimum. A cleaner and larger source (especially buyers, not all visitors) = better.
- **Threshold:** Meta recommends **1,000-5,000**; some agencies go up to **5,000-20,000**. This isn't a contradiction: event quality matters more than volume.
- **Reliability:** [OFFICIAL META] (1,000-5,000) + [AGENCY CONSENSUS]

### Rule 2.7: Custom audiences / retargeting — windows
- **Gist:** Match the window to the purchase cycle: short (3-7 days) for hot leads, long (30-90) for reactivation.
- **Retention limits [OFFICIAL META]:** Website Custom Audience — up to 180 days (up to 2 years for purchase events); FB/IG/video/event engagement — up to 365 days.
- **Working windows [AGENCY CONSENSUS]:** website visitors often 90 days; frequent-purchase 28-30; video segments — viewers who watched 50%/75%.
- **Reliability:** [OFFICIAL META] (limits) + [AGENCY CONSENSUS] (windows)

### Rule 2.8: Detailed targeting (interests) — nearly dead
- **Gist:** Interests as a foundation are outdated. Where they still make sense (as a seed/hint): new accounts with few conversions, very niche/hyperlocal markets, small budgets (<$30/day). Even then — as a hint inside A+A.
- **Reliability:** [AGENCY CONSENSUS]

### Rule 2.9: Audience size
- **Gist:** An audience that's too narrow chokes delivery — a common cause of getting stuck in learning.
- **Threshold:** for conversions — a potential reach of 1M+; below 500k the risk of Learning Limited rises sharply. For broad/A+A there's no upper bound — the algorithm narrows it on its own.
- ⚠️ These reach numbers depend on geo. For small markets, 1M may be unattainable — see file 06.
- **Reliability:** [AGENCY CONSENSUS]

### Rule 2.10: Audience overlap (ALIGNED with file 01)
- **Gist:** Overlap → auction competition → more expensive. Check with the Audience Overlap Tool (works for audiences of 10k+). Don't test each interest in a separate ad set.
- **Threshold:** target **<15-20%**; act when **>25-30%**.
- **Reliability:** [OFFICIAL META] (the tool) + [AGENCY CONSENSUS] (the thresholds)

### Rule 2.11: Common mistakes
- Hyper-narrow interest stacks → small reach, slow learning.
- Too many fragmented ad sets (5 ad sets = you need 250 conversions for them all to exit learning).
- A budget that's too small relative to CPA.
- Duplicating warm and cold audiences without exclusions.

### Rule 2.12: Creative beats audience (the Andromeda shift)
- **Gist:** With the Andromeda AI engine, Meta moved from "audience-first" to "creative-first." With broad targeting, the media buyer's role shifts toward producing many creatives (see file 03).
- **Reliability:** [AGENCY CONSENSUS] (Andromeda is Meta's official name)

---

## CONTRADICTIONS AND OPEN QUESTIONS

1. **Are audience hints prioritized?** Meta: yes. Independent tests (Loomer): no. → a weak lever.
2. **Which Lookalike %?** "Always 1%" (outdated) vs "2-5%, test it" (current).
3. **Do you need manual LALs/custom audiences with A+A?** "Everything on broad" vs "first-party data is still the best foundation."

## OUTDATED — DO NOT APPLY (with exact dates)

- **Detailed targeting exclusions** — being phased out: from **March 31, 2025**, banned in new ad sets; from **June 10, 2025**, removed from boosted posts; legacy campaigns run until **January 15, 2026**, then stop.
- **Granular interests** (EDM fans, SUVs, etc.) — as of **June 23, 2025**, consolidated into broad groups.
- A stack of dozens of interests in one ad set "for precision" — fragments the signal.
- A separate ad set for each interest to test it — causes overlap.
- "1% Lookalike is always optimal" — an outdated dogma.
- Manual LALs/interests as the core of targeting — replaced by broad + A+A.
